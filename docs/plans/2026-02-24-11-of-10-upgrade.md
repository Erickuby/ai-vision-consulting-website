# 11/10 Website Upgrade Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Transform the AI Vision Consulting website from a solid B+ into a genuinely spectacular 11/10 experience with real 3D hero, custom cursor, page loader, scroll progress, sticky CTA, social proof ticker, real contact form backend, clean hooks, and data extraction.

**Architecture:** React 19 + Vite + Three.js/R3F already installed. NeuralCanvas already uses R3F for the background. We extend it with a dedicated hero 3D scene (two-column layout), add UX polish components, extract repeated logic into hooks, move hardcoded data to src/data/, and wire the contact form to EmailJS.

**Tech Stack:** React 19, TypeScript, Framer Motion 12, Three.js 0.183, @react-three/fiber 9, @react-three/drei 10, Tailwind CSS 4, Vite 7, EmailJS (free tier), Lucide React
